## Stack and Queue
### Stack
- Last in First out (LIFO)
- no fixed size
- Push to add, Pop to return value and remove
- Peek -> see last
- Think of stack of books 
- Ex: print array in reverse
### Queue
- First in First out
- No fixed size
- Offer to add, Poll to return value and remove
- Used in buffer for keyboard
