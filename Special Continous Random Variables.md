# Uniform
> ### Definition
>- A Random variable that is uniformly distributed over an interval [a,b]
>- Constant PDF
>- f(x) -> 1/(b-a) where a <= x <= b
> ### Mean
>- E(x) =  M = (a+b)/2
> ### Variance
>- (a-b)^2/12
